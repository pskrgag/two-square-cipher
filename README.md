# Two square cipher
Simple Two square cipher visualization

# Requirements 
  - Qt5
  - Cmake 2.8 or higher
# Build and run
  ```sh
$ git clone https://github.com/pskrgag/two-square-cipher.git
$ cd two-square-cipher
$ mkdir build && cd build
$ cmake .. && make 
$ ./two-square-cipher
```
